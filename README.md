# SimpleForth

SimpleForth compiles Forth-like language to JavaScript.

## Installation

Via npm on Node:

```
npm install simpleforth
```


## Usage

Reference in your program:

```js
var sf = require('simpleforth');
```

## Development

```
git clone git://github.com/ajlopez/SimpleForth.git
cd SimpleForth
npm install
npm test
```
